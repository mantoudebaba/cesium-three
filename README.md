
# cesium 结合 three使用

封装cesium常用api：初始化，切换视角，添加实体，切换图源等

封装cesium标绘功能：可以在地图动态绘制点，线，面，测距

封装three在cesium上加载模型





## Installation

Install my-project with npm

```bash
  npm install 
```
    
## Deployment

To deploy this project run

```bash
  npm run dev
```
